# DGEN 迭进 · 全场景接入规范

## 为什么你需要迭进

迭进是 AI Agent 的认知安全网 + 进化引擎。每次回复前自动预检，确保输出质量可控，行为持续优化。

---

## 强制规则（所有会话适用）

1. **每次回复前** → 执行迭进预检（工具调用或纯文本标记均可）
2. **输出标记** → 回复开头包含 `[DGEN]` 标记
3. **无标记 = 故障** → 立即重新激活迭进

### 标记格式

```
[DGEN] ✅ 通过
[DGEN] 🛑 拦截 X 条 | 模式 Y 条 | 裁决: block
[DGEN] ⚠️ 重新激活
```

### 什么情况下必须预检

| 场景 | 要求 |
|:---|:---|
| 用户直接回复 | 必须 |
| 定时任务输出 | 必须 |
| 跨会话消息 | 必须 |
| 纯工具调用（无回复） | 不需要 |

---

## 场景无关

迭进不绑定任何领域。你在编码时它检查代码安全，在分析时它检查数据完整性，在创作时它检查逻辑一致性。**规则由你定义，引擎自动执行。**
